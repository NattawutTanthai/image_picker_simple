## How to use ?
```shell
git clone https://github.com/NattawutTanthai/image_picker_simple.git
```
```shell
cd image_picker_simple
```
```shell
npm install
```
open metro : 
```shell
npm start
```
and :
```shell
npm run android
```
or :
```shell
npm run ios
```
options or orther settings [click here](https://www.npmjs.com/package/react-native-image-picker)
